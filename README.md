# AWS Ansible Automation

## Project Overview

This project demonstrates infrastructure automation using AWS EC2 instances and Ansible.

## Technologies Used

- AWS EC2
- Ansible
- Ansible Navigator
- Docker
- Git & GitHub
- Apache Web Server

## Project Tasks

### Task 1 - AWS Infrastructure Setup
- Created control and managed nodes
- Configured SSH connectivity

### Task 2 - Ansible Environment Setup
- Installed Ansible Navigator
- Configured inventory and ansible.cfg

### Task 3 - Package Management
- Installed PHP and MariaDB packages
- Managed package installation using playbooks

### Task 4 - Roles and Templates
- Created Ansible role
- Configured Apache Web Server
- Deployed dynamic web page using templates

### Task 5 - Content Modification
- Updated /etc/issue file for different host groups

### Task 6 - Custom Web Server Configuration
- Created custom document root
- Configured symbolic links
- Deployed development web page
